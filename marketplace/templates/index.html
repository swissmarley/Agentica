Manual Upload

How to use with Agentica CLI

1. Pull an Agent (Install)

Download an agent from the marketplace into your local environment.

agentica pull <agent_name> --version <version>

2. Push an Agent (Publish)

Upload your local agent bundle to the marketplace.

agentica push path/to/agent.zip

Note: Ensure your agentica_manifest.json is valid before pushing.
